# Moshi Downloader


**Moshi Downloader** é um programa CLI simples para baixar vídeos/áudios.
Plataformas testadas:
- [x] YouTube
- [x] Instagram
- [ ] Outros...
<br>

Está disponível para:
- [x] Termux
- [x] Linux
- [x] MacOS
- [x] Windows

<br>

Para usar o Moshi, siga os seguintes passos:
1. Instale o binário de acordo com sua plataforma
2. Entre na pasta onde o binário está instalado
3. Use `./{program}` com o nome do seu binário (exemplo: `./moshi_linux`)
4. O resto está explicado no código  :D

<br>O projeto ainda está em fase de **testes**, portanto, bugs são "normais".
<br>Moshi usa o sistema cross-compiling do Go, então se o programa **não** rodar na sua máquina, peço mil perdões 🙏🏻
<br>Por enquanto, está apenas disponível em português, mas em breve irei colocar suporte à outros idiomas :)
